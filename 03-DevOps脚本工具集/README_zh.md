# DevOps 脚本工具集

本仓库包含 1000+ 个 Bash 脚本和配置文件，覆盖云平台、CI/CD、数据库、Docker、Kubernetes、监控等自动化场景。

---

## 📁 **主要目录**

| 目录 | 用途 |
|------|------|
| ai-AI工具 | AI 相关脚本 |
| aws-AWS云平台 | AWS 自动化脚本 |
| gcp-GCP云平台 | Google Cloud 脚本 |
| azure_devops-Azure-DevOps | Azure DevOps 脚本 |
| docker-Docker | Docker 管理脚本 |
| kubernetes-Kubernetes | K8s 自动化脚本 |
| jenkins-Jenkins | Jenkins CI 脚本 |
| terraform-Terraform | 基础设施即代码脚本 |
| monitoring-监控工具 | 监控相关脚本 |
| configs-配置文件 | 各类工具配置文件 |

---

## 🚀 **如何使用**

1. 根据需求选择相应目录
2. 查看脚本文件了解功能
3. 按照注释说明使用脚本

---

## 📚 **学习建议**

- 先了解 Bash 脚本基础
- 从简单脚本开始学习
- 逐步尝试复杂脚本
