# Ansible 自动化示例

《Ansible for DevOps》书籍配套代码，按章节组织的 Ansible Playbook 示例集。

---

## 📁 **目录结构**

| 目录 | 用途 |
|------|------|
| first-ansible-playbook-首个Playbook | 基础 Playbook |
| drupal-Drupal部署 | Drupal 部署 |
| nodejs-Node.js部署 | Node.js 部署 |
| kubernetes-Kubernetes集群 | K8s 集群部署 |
| docker-Docker管理 | Docker 管理 |

---

## 🚀 **如何使用**

1. 选择一个示例目录
2. 查看 README 了解用法
3. 运行 ansible-playbook 命令
