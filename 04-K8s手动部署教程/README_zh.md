# K8s 手动部署教程

手把手从零搭建 Kubernetes 集群的教程，深入理解 etcd、API Server、Worker Node 等组件原理。

---

## 📁 **目录结构**

| 目录 | 用途 |
|------|------|
| configs-配置文件 | K8s 组件配置模板 |
| docs-文档教程 | 13 步部署教程文档 |
| units-服务单元 | systemd 服务文件 |

---

## 🚀 **如何使用**

1. 按照 docs-文档教程 中的步骤顺序执行
2. 使用 configs-配置文件 中的模板配置组件
3. 使用 units-服务单元 中的服务文件启动服务
