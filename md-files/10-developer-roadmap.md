## 开发

克隆仓库，安装依赖并启动应用

```bash
git clone git@github.com:nilbuild/developer-roadmap.git --depth 1
cd developer-roadmap
pnpm add @roadmapsh/editor@npm:@roadmapsh/dummy-editor -w
pnpm install
```

使用以下命令启动开发服务器：

```bash
pnpm dev
```
> 运行应用前，请确保将 `.env.example` 重命名为 `.env`。
