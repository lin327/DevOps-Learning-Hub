# Flux2 GitOps 示例

FluxCD 官方示例，演示使用 Flux v2 + Kustomize + Helm 实现 GitOps 风格的 Kubernetes 集群管理。

---

## 📁 **目录结构**

| 目录 | 用途 |
|------|------|
| apps-应用 | Helm 应用定义 |
| clusters-集群 | 集群配置 |
| infrastructure-基础设施 | 共享基础设施 |
| scripts-脚本 | 验证脚本 |

---

## 🚀 **如何使用**

1. 查看 apps-应用 中的应用定义
2. 了解 clusters-集群 中的集群配置
3. 学习 infrastructure-基础设施 中的基础设施配置
