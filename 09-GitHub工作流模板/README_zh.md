# GitHub 工作流模板

GitHub 官方 Actions 启动工作流模板库，包含 CI、代码扫描、部署、Pages 等模板。

---

## 📁 **目录结构**

| 目录 | 用途 |
|------|------|
| automation-自动化工作流 | 自动化任务模板 |
| ci-持续集成 | CI 工作流模板 |
| code-scanning-代码扫描 | 安全扫描模板 |
| deployments-部署工作流 | 部署模板 |
| pages-Pages部署 | Pages 部署模板 |

---

## 🚀 **如何使用**

1. 选择适合的工作流模板
2. 复制到 .github/workflows 目录
3. 根据需求修改配置
