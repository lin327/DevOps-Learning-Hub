# 开发者学习路线图

roadmap.sh 网站源码，提供交互式开发者学习路线图，涵盖前端、后端、DevOps、AI 等方向。

---

## 📁 **目录结构**

| 目录 | 用途 |
|------|------|
| src-源代码 | Astro + React 应用源码 |
| public-静态资源 | 图片、字体、路线图数据 |
| scripts-构建脚本 | 内容生成脚本 |
| packages-子包 | monorepo 子包 |

---

## 🚀 **如何使用**

1. 安装依赖: pnpm install
2. 启动开发服务器: pnpm dev
3. 访问 http://localhost:3000
