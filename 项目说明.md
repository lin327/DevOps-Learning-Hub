# GitHub 仓库项目总览

本目录包含 11 个精选的开源项目，涵盖 DevOps、容器化、监控、CI/CD、自动化等技术领域，以及 1 个中文文档翻译集合。

---

## 目录结构与说明

| 序号 | 文件夹名称 | 用途说明 |
|------|-----------|---------|
| 01 | **01-DevOps练习题库** | 包含 2624 道 DevOps/SRE 面试题和练习，涵盖 Linux、Docker、Kubernetes、AWS、网络等 42 个技术主题 |
| 02 | **02-DevOps实战项目** | 40 个从入门到高级的 DevOps 实战项目，涵盖 AWS、Terraform、Jenkins、Docker、Kubernetes、ArgoCD 等 |
| 03 | **03-DevOps脚本工具集** | 1000+ 个 Bash 脚本和配置文件，覆盖云平台、CI/CD、数据库、Docker、Kubernetes、监控等自动化场景 |
| 04 | **04-K8s手动部署教程** | 手把手从零搭建 Kubernetes 集群的教程，深入理解 etcd、API Server、Worker Node 等组件原理 |
| 05 | **05-Flux2-GitOps示例** | FluxCD 官方示例，演示使用 Flux v2 + Kustomize + Helm 实现 GitOps 风格的 Kubernetes 集群管理 |
| 06 | **06-Prometheus监控方案** | Docker Compose 部署的 Prometheus + Grafana 监控栈，包含 cAdvisor、node-exporter、Alertmanager |
| 07 | **07-Docker-ELK日志系统** | Docker Compose 部署的 Elasticsearch + Logstash + Kibana 日志收集分析平台 |
| 08 | **08-Ansible自动化示例** | 《Ansible for DevOps》书籍配套代码，按章节组织的 Ansible Playbook 示例集 |
| 09 | **09-GitHub工作流模板** | GitHub 官方 Actions 启动工作流模板库，包含 CI、代码扫描、部署、Pages 等模板 |
| 10 | **10-开发者学习路线图** | roadmap.sh 网站源码，提供交互式开发者学习路线图，涵盖前端、后端、DevOps、AI 等方向 |
| 11 | **11-从零构建技术指南** | 精选教程集合，教你从零实现操作系统、数据库、编译器、Web 服务器等 28 类技术项目 |
| 12 | **md-files/中文文档翻译** | 上述项目的中文翻译文档集合，每个项目对应一个 Markdown 文件 |

---

## 命名规则

所有文件夹采用 `英文原名-中文说明` 的格式，例如：
- `images-图片资源`
- `topics-主题分类`
- `aws-AWS云服务`
- `exercises-练习`

## 使用建议

- **面试准备**: 重点看 `01-DevOps练习题库` 和 `02-DevOps实战项目`
- **技能提升**: 按 `10-开发者学习路线图` 规划学习路径
- **动手实践**: 从 `04-K8s手动部署教程` 和 `07-Docker-ELK日志系统` 开始
- **自动化运维**: 学习 `03-DevOps脚本工具集` 和 `08-Ansible自动化示例`
- **中文阅读**: 查看 `md-files/中文文档翻译` 文件夹中的翻译文档
